# EBC-Formula

## ${EBC}_{total} = \frac{t_{boiling}}{60} + \frac{Plato}{10} * {\sum_{i=1}^n{x_i * {EBC}_i}}$

- $t_{boiling}$ [min]
- $Plato$ [°P]
- $x_i$ [%]


## ${EBC}_{total} = \frac{t_{boiling}}{60} + \frac{Plato}{10} * \frac{\sum_{i=1}^n{m_i * {EBC}_i}}{m_{total}}$

- $t_{boiling}$ [min]
- $Plato$ [°P]
- $m_i$ [kg]
- $m_{total}$ [kg]
